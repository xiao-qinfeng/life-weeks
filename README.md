# 生命周数可视化工具

一个遵循迪特·拉姆斯简约设计哲学的生命周数可视化工具，帮助用户直观地了解自己的人生进度。

## 功能特性

- 📊 **生命周数网格**: 以90年为基准，动态计算并显示4,680周的生命总长度
- 🎨 **macOS系统配色**: 采用macOS原生色彩 - 红色(已过)、绿色(剩余)、黄色(当前周)
- 💭 **反思性文字**: 提供关于时间价值的深刻思考
- 🌓 **双主题模式**: 支持浅色和深色主题切换
- 📱 **社交分享**: 生成适合社交媒体分享的卡片格式
- 🖼️ **壁纸导出**: 一键导出为PC、手机和社交媒体的4K高清壁纸
- 📏 **响应式设计**: 方块自动调整大小，填满不同尺寸的屏幕
- 🎯 **精准定位**: 高亮显示当前所在的周数位置

## 技术栈

- React 18 (Hooks)
- HTML2Canvas (用于图片导出)
- Flexbox 布局 (响应式设计)
- 纯CSS样式 (遵循极简设计原则)

## 安装和运行

### 前提条件
- Node.js (版本 14 或更高)
- npm 或 yarn

### 安装步骤

1. 克隆或下载项目
2. 安装依赖：
   ```bash
   npm install
   ```
3. 启动开发服务器：
   ```bash
   npm start
   ```
4. 打开浏览器访问 `http://localhost:3000`

### 构建生产版本

```bash
npm run build
```

## 使用方法

1. **输入出生日期**: 选择您的出生日期，系统将自动计算已度过和剩余的周数
2. **切换主题**: 点击主题切换按钮(☀️/🌙)在浅色和深色模式间切换
3. **查看可视化**:
   - 网格显示：每个小方块代表一周，红色表示已度过，绿色表示剩余
   - 黄色方块高亮显示当前所在的周数位置
   - 方块自动调整大小，填满整个屏幕空间
4. **分享和导出**:
   - 点击"分享"按钮分享到社交媒体
   - 使用"PC"和"手机"按钮导出不同尺寸的壁纸
   - 导出图片占据85-90%的可用空间，白边最小化

1. **好的设计是创新的** - 采用现代化的交互方式

3. **好的设计是美观的** - 简洁优雅的视觉呈现
4. **好的设计让产品易于理解** - 直观的周数可视化
5. **好的设计是不唐突的** - 不打扰用户的思考
6. **好的设计是诚实的** - 真实反映时间流逝
7. **好的设计是持久的** - 经典的设计风格
8. **好的设计关注每一个细节** - 精心设计的交互体验
9. **好的设计是环保的** - 轻量级的代码实现
10. **好的设计是尽可能少的设计** - 去除不必要的元素
